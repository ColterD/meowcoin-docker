Platform Status SETUP MODE

The platform is currently running in setup mode with simplified services. Follow the instructions below to enable full functionality.

Getting Started

To enable the full functionality of the MeowCoin Platform, follow these steps:

  1. Review and update the .env file with your preferred settings
  2. Run the setup script: ./setup.sh
  3. Switch to the full docker-compose file: mv docker-compose.full.yml docker-compose.yml
  4. Restart the platform: docker-compose down && docker-compose up -d

Available Services

  • Dashboard: http://localhost:3000
  • Grafana: http://localhost:3001 (default credentials: admin/admin)
  • RabbitMQ Management: http://localhost:15672 (default credentials: guest/guest)
  • PostgreSQL: localhost:5432 (default credentials: postgres/postgres)
  • Redis: localhost:6379

Important Security Notice

Default credentials have been set for demonstration purposes. For production use, please update the passwords in the .env file.

Particularly important to change:

  • JWT_SECRET
  • MEOWCOIN_RPC_PASSWORD
  • POSTGRES_PASSWORD
  • GRAFANA_ADMIN_PASSWORD
  • RABBITMQ_PASSWORD
